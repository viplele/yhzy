隐私策略

1. 我们收集的信息

网站：无

DNS 服务器：无

DoT 服务器：无

DoH 服务器：无

我们不会记录访问我们 DoH 服务器的用户的个人身份信息 （PII），包括用户 IP、浏览器信息等。这些日志在服务器上被禁用。

我们应用 DNS 查询名称最小化，以最大限度地减少从 DNS 解析器发送到权威名称服务器的数据量，以确保和提高隐私安全性。我们特此声明，我们的 DoH 服务符合 RFC 7816 并且可以用于任何级别的 Internet 标准。

2. 我们共享的信息

网站：无

DNS 服务器：无

DoT 服务器：无

DoH 服务器：无

我们保留出于法律原因阻止您的 IP 地址和/或 ASN 的权利，包括但不限于：

  • 使用我们的 DNS 服务时滥用我们的网络。
  • 调查潜在的违规行为服务条款。
  • 保护我们的安全以及其他用户和我们的权利免受损害。
  • 检测、预防、处理或调查欺诈、安全或技术问题。
  • 满足任何适用的法律、法规、法律程序或强制性的政府要求。

    • 我们不能分享我们没有、未存储或之前已删除的任何信息。

    3. 饼干

    目前，我们没有在我们的网站或 DNS 服务中积极使用 Cookie。

    4. 第三方服务

    目前，我们不在我们的网站或 DNS 服务中使用任何第三方服务。

    5. 变更

    我们的隐私政策可能会不时更改。如有更改，我们将在我们的网站上发布通知。

    6. 服务终止

    您可以选择终止使用我们的服务，并写信给我们要求删除您的数据。请注意，这只是尽力而为，因为大多数数据都是匿名的，我们可能无法找到与您相关的所有数据。

    请随时发送电子邮件，或邮寄法律信函至 lele@krutt.org
